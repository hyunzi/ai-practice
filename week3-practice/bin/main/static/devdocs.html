DevDocs RAG 테스트

적재 → 검색 → 에이전트 → LLM → Chroma 점검을 한 화면에서 확인하세요.

1) DevDocs 적재 (기본 size300)

`/api/devdocs/ingest`로 마크다운을 chunk/임베딩 후 스토어에 넣습니다.

아직 실행 전입니다.

2) RAG 검색

`/api/devdocs/query`로 size300/size600 컬렉션을 조회합니다.

아직 실행 전입니다.

3) 에이전트에게 묻기

`/api/agent/ask` (필요 시 RAG 툴 사용). 간단한 베이스라인 에이전트입니다.

아직 질문 전입니다.

4) Raw LLM 호출 (RAG 없이)

`/api/openai/chat`으로 모델만 직접 호출합니다.

아직 호출 전입니다.

5) Chroma 컬렉션 확인

컬렉션 목록/프리뷰로 스토어 상태를 살펴봅니다. Chroma 서버가 없으면 인메모리 결과가 비어있을 수 있습니다.

아직 조회 전입니다.
아직 미리보기 전입니다.
